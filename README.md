# 데이터전처리(Data Preprocessing)
- 데이터 전처리 수업은 데이터를 특정 모델에 적용해 분석하기 전 다양한 전처리 방법을 배우고, 이를 새로운 문제에 적용하는 데 중점을 둔 수업입니다. 수업에서는 데이터에 입각한 결정을 내리기 위해 데이터의 구조 및 전처리 방법을 이해합니다.
